# Siren Multivendor Search  
### One Platform. 250+ Intelligence Sources. Zero Switching.

## Overview

Siren Multivendor Search allows investigators to access and query over 250 external intelligence and data vendors — directly from within the Siren platform.

There is no need to log into multiple systems or switch between vendor portals.

Search once. Query everywhere.

---

## External Intelligence, Unified

Siren connects to a wide range of external data sources, including:

- Dark web intelligence  
- OSINT providers  
- News and media archives  
- Cryptocurrency intelligence  
- Corporate registries  
- Sanctions and watchlists  
- Identity databases  
- Web and domain intelligence  

And many more.

The platform is not limited to a fixed list — new vendor integrations can be added as needed.

---

## How It Works

Investigators submit a single search.

Siren securely queries connected vendors and aggregates results into one unified workspace.

Results can then be:

- Correlated with internal case data  
- Linked to existing entities  
- Visualized in network analysis  
- Included in investigative reports  

This removes operational friction and accelerates intelligence gathering.
