# Entity Tagger

The **Entity Tagger** provides analysts with a simple interface to review and refine entities extracted from documents using AI. It allows users to process individual documents through entity extraction and then interactively review the results.

Analysts can easily add new entities, modify existing ones, or remove incorrect extractions. The interface is designed to support quick human validation of AI-generated results, improving overall data quality and accuracy.

The tool also includes lightweight workflow tracking, allowing documents to move through simple stages such as *Draft*, *In Progress*, and *Done*.

## Key Capabilities

- Review AI-extracted entities through an intuitive interface
- Process individual documents for entity extraction
- Add, edit, or delete detected entities
- Improve accuracy with human-in-the-loop validation
- Track document review status with simple workflow stages
