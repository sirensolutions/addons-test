## OSINT Data Ingestion & Monitoring

Siren OSINT supports large-scale, continuous ingestion of open-source data via integrated OSINT collection platforms.

**Project-Based Monitoring**
- Projects define investigation themes (e.g. protests, extremism, disinformation, fraud)
- Topics include keywords, hashtags, entities, locations, and account handles
- Each project groups multiple collection jobs into a single monitoring context

**OSINT Collection**
- Integration with platforms such as **Fivecast** and **ShadowDragon**
- Periodic collection of:
  - Social media posts
  - New and updated accounts
  - Engagement and metadata signals
- Sources include X, Facebook, forums, messaging platforms, and other open sources

**Ingestion Pipeline**
- Automated Logstash pipelines retrieve collected data
- Content is processed, normalised, and enriched
- Data is indexed into **Elasticsearch** and made available in **Siren Investigate**
